PCA¶

Using prcomp is an easier way than the one we have learnt in the meeting to do PCA(learnt from the ISL book). When I do PCA, the prcomp function by default centers the data but not necessarily scales the data. I have to try both to see which one performs better.

In [6]:
pr.out <- prcomp(gene_exp)
